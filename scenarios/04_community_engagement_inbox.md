# Community Engagement Inbox Triage

- Domain: community engagement and public feedback.
- Build prompt: Triage community comments into themes, urgency, and suggested next action.
- Sample inputs: `sample_data/community_comments.csv`.
- MVP requirements: deterministic keyword rules, urgency tags, human review queue.
- Stretch goals: deduping, sentiment assist, response drafting, CRM sync.
- Production-grade signals: human-in-the-loop, no automated sensitive replies, clear reason codes.
- Codex practice loop: start with rules, test edge cases, summarize limitations.

After completing this scenario: run tests, summarize the diff, write a demo note, run the retrospective prompt, and decide whether exactly one harness improvement should be promoted.
