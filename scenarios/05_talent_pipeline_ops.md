# Talent Pipeline Ops Agent

- Domain: people operations and recruiting.
- Build prompt: Build a pipeline tracker that flags stalled candidates and next actions.
- Sample inputs: create a small CSV based on the launch task shape or extend sample data.
- MVP requirements: stage counts, stale candidate detection, owner reminders.
- Stretch goals: interview load balancing, calendar handoff, scorecard completeness.
- Production-grade signals: fairness awareness, audit trail, no fully automated reject decisions.
- Codex practice loop: model states clearly, implement stale logic, verify with sample rows.

After completing this scenario: run tests, summarize the diff, write a demo note, run the retrospective prompt, and decide whether exactly one harness improvement should be promoted.
